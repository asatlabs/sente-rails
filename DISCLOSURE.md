# Disclosure Build

This repository is a **redacted, read-only disclosure build** of Sente Rails,
published so that authorised reviewers can inspect the system.

It is **not** the complete source. The authoritative data model and selected
operational material have been withheld from this build **and its history**:

- database schema / data-model definitions
- the machine-readable API contract (OpenAPI specification)
- the full architecture specification
- deployment, infrastructure, and migration material

Active development continues in a separate private repository. This build is a
point-in-time disclosure snapshot and is not maintained for production use.
